# 1. Vision

AI Engineering Framework is a universal, technology-independent engineering methodology for developing, modernizing and maintaining software products with the assistance of Artificial Intelligence.

Its central objective is to preserve and continuously evolve Engineering Knowledge while enabling predictable, secure and maintainable software development.

---

# 2. Development Strategy

The Framework is developed incrementally.

Core rules:

- Knowledge First.
- Product First.
- Structure before details.
- Architecture before implementation.
- Introduce process only when complexity requires it.
- One logical step equals one commit.
- Every completed step leaves the repository stable.
- Practical validation precedes universal adoption.
